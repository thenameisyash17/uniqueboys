Q1: What is the difference between 4-connectivity and 8-connectivity?

4-connectivity: Considers only horizontal and vertical neighbors (up, down, left, right).

8-connectivity: Includes diagonal neighbors in addition to 4-connectivity.

When to use: 4-connectivity for simple shapes, 8-connectivity for complex boundaries.

Q2: Compare Sobel, Prewitt, and Canny edge detectors

Sobel: Uses 3x3 gradient approximation, good for horizontal/vertical edges.

Prewitt: Simpler than Sobel, more sensitive to noise.

Canny: Multi-stage algorithm with noise reduction, gradient calculation, non-max suppression, hysteresis.

Best Practice: Canny for most applications, Sobel for simple edge detection.

Q3: Why is SIFT scale and rotation invariant?

Scale Invariance: Uses Difference of Gaussian (DoG) across scale space pyramid.

Rotation Invariance: Computes dominant orientation based on gradient histogram.

Key Steps: Scale-space extrema detection, keypoint localization, orientation assignment, descriptor generation.

Applications: Object recognition, image stitching, 3D reconstruction.

Two-stage (R-CNN, Faster R-CNN):

1. Region proposal generation

2. Classification & bounding box regression

Single-stage (YOLO, SSD):

1. Direct prediction of classes and boxes

2. Faster but potentially less accurate

Trade-off: Speed vs accuracy

Fundamentals

Q5: What is camera calibration and why is it important?

Definition: Process of estimating camera intrinsic and extrinsic parameters.

Intrinsic Parameters: Focal length, principal point, skew, distortion coefficients.

Extrinsic Parameters: Camera position and orientation in world coordinates.

Importance: Essential for 3D reconstruction, stereo vision, AR/VR, robotics.

Core Concept 4 min read

Image Processing

Q6: Compare spatial vs frequency domain filtering

Spatial Domain: Direct pixel manipulation, convolution with kernels.

Frequency Domain: Fourier transform, filtering in frequency space.

Advantages Spatial: Simple, fast for small kernels.

Advantages Frequency: Better for global operations, theoretical insights.

Technical Comparison 3 min read

Feature Detection

Q7: What makes a good feature descriptor?

Distinctiveness: Unique representation for different features.

Robustness: Invariant to illumination, scale, rotation.

Efficiency: Fast to compute and match.

Compactness: Low memory footprint.

Examples: SIFT (distinctive), ORB (efficient), BRIEF (compact).

Design Principles 4 min read

Deep Learning

Q8: How does CNN achieve translation invariance?

1. Weight Sharing: Same filters applied across image.

2. Pooling Layers: Reduce spatial dimensions, aggregate features.

3. Hierarchical Features: Low-level ‚Üí high-level abstraction.

4. Data Augmentation: Training with translated examples.

Architecture Insight 3 min read

3D Vision

Q9: What is the difference between sparse and dense reconstruction?

Sparse Reconstruction: 3D points only at detected features, fast, used in SLAM.

Dense Reconstruction: 3D points for every pixel, detailed, used in 3D modeling.

Applications: Sparse for navigation, dense for modeling and visualization.

Trade-off: Accuracy vs computational cost.

Applications

Q10: What are challenges in real-time computer vision?

1. Computational Speed: Processing high-resolution video in real-time.

2. Accuracy vs Speed: Trade-off between algorithm complexity and performance.

3. Environmental Factors: Lighting changes, weather conditions.

4. Hardware Constraints: Power consumption, memory limitations.
