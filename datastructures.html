Data Structures Cheatsheet

Arrays

  • Access by Index O(1)
  • Search O(n)
  • Insertion (end) O(1)
  • Insertion (middle) O(n)
  • Deletion O(n)

Contiguous memory, fixed size, cache-friendly

Linked Lists

  • Access O(n)
  • Search O(n)
  • Insertion (head) O(1)
  • Insertion (middle) O(n)
  • Deletion (head) O(1)

Dynamic size, non-contiguous memory

Binary Search Trees

  • Search (avg) O(log n)
  • Search (worst) O(n)
  • Insertion (avg) O(log n)
  • Deletion (avg) O(log n)
  • Traversal O(n)

Left < parent < right property

Hash Tables

  • Search (avg) O(1)
  • Search (worst) O(n)
  • Insertion (avg) O(1)
  • Deletion (avg) O(1)
  • Space O(n)

Handles collisions with chaining/open addressing

Sorting Algorithms

  • Bubble Sort O(n²)
  • Selection Sort O(n²)
  • Insertion Sort O(n²)
  • Merge Sort O(n log n)
  • Quick Sort (avg) O(n log n)
  • Heap Sort O(n log n)

Search Algorithms

  • Linear Search O(n)
  • Binary Search O(log n)
  • Depth-First Search O(V+E)
  • Breadth-First Search O(V+E)
  • Dijkstra's Algorithm O(E log V)
