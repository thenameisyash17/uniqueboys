Q1: What is the difference between AI, ML, and DL?

AI (Artificial Intelligence): Broad field of creating intelligent machines.

ML (Machine Learning): Subset of AI that enables machines to learn from data without explicit programming.

DL (Deep Learning): Subset of ML using neural networks with multiple layers to learn representations of data.

Q2: Compare Batch, Mini-batch, and Stochastic GD

Batch GD: Uses entire dataset for each update. Stable convergence but slow for large datasets.

Mini-batch GD: Uses small random subsets. Best trade-off between speed and stability.

Stochastic GD: Uses single sample per update. Fast but noisy convergence.

Q4: Why do transformers outperform RNNs for NLP?

1. Parallelization: Transformers process all tokens simultaneously vs sequential RNNs.

2. Long-range Dependencies: Self-attention captures relationships regardless of distance.

3. Gradient Flow: No vanishing gradient problem in deep networks.

4. Context Understanding: Bidirectional context in BERT vs unidirectional in RNNs.

Q5: What is the key innovation in ResNet?

Residual Connections: Skip connections that allow gradients to flow directly through layers.

Problem Solved: Vanishing gradients in very deep networks.

Key Equation: \( H(x) = F(x) + x \) where F(x) is residual mapping

Impact: Enabled networks with 100+ layers (ResNet-152).

Fundamentals

Q6: Explain the bias-variance tradeoff in deep learning

Bias: Error from erroneous assumptions in learning algorithm. High bias causes underfitting.

Variance: Error from sensitivity to small fluctuations in training set. High variance causes overfitting.

Tradeoff: As model complexity increases, bias decreases but variance increases.

DL Solution: Large networks have low bias; regularization controls variance.

CNN Questions

Q7: What is the purpose of padding in convolutional layers?

1. Size Preservation: 'Same' padding maintains input spatial dimensions.

2. Border Information: Prevents loss of information at edges.

3. Kernel Application: Allows kernel to be applied to border pixels.

4. Control Output Size: Controls output feature map dimensions.

Technical Detail 2 min read

RNN Questions

Q8: How does LSTM solve vanishing gradient problem?

1. Cell State: Constant error carousel allows gradient flow.

2. Gates: Forget gate controls what to remember/forget.

3. Additive Updates: Gradient doesn't vanish through addition.

4. Selective Memory: Learns what information to preserve.

Training Questions

Q9: Compare Adam vs SGD with momentum

Adam: Adaptive learning rates per parameter, combines momentum and RMSprop.

SGD+Momentum: Fixed learning rate with velocity accumulation.

When to use Adam: Default choice for most problems.

When to use SGD: When you need reproducible results or better generalization.

Transformer Questions

Q10: What is self-attention and how is it calculated?

Definition: Mechanism to compute representation of sequence by attending to all positions.

Calculation: \( \text{Attention}(Q,K,V) = \text{softmax}(\frac{QK^T}{\sqrt{d_k}})V \)

Q, K, V: Query, Key, Value matrices from input.

Scale factor: \( \sqrt{d_k} \) prevents small gradients.

Ethics & Limitations

Q11: What are major limitations of current DL?

1. Data Hungry: Requires large labeled datasets.

2. Black Box: Lack of interpretability.

3. Computationally Expensive: High GPU requirements.

4. Brittle: Vulnerable to adversarial attacks.

5. Bias Amplification: Can amplify societal biases.
